Главные выводы

  1. Слабо занятая ниша — полностью автоматизированный 360-анализ бизнеса (финансы + конкуренты + HR + digital + стратегия) в интерактивном HTML-формате не предлагает ни один игрок; AI.Инсайт (Контур) ближе всех, но фокусируется на проверке контрагентов
  2. Пустое пространство — между бесплатным промптом в ChatGPT (глубина 1-2/10) и консалтингом за 500K+ руб. (глубина 7-9/10) нет промежуточного продукта
  3. Вакуум после Big4/Big3 — McKinsey, BCG, PwC, Deloitte ушли; рынок консалтинга вырос на 15% и ищет новых провайдеров
  4. AI-платформы на раннем этапе — AI.Инсайт (Контур) уже запущен, СПАРК и Контур.Фокус добавляют AI-функции, но комплексный стратегический анализ 360 пока не покрыт
  5. Главная угроза — Сбер/Яндекс/Контур могут войти в нишу с несравнимыми ресурсами и данными
  6. Главная слабость BSR — качество 5/10; без доведения до 7+/10 продукт не конкурентоспособен

Ключевой разрыв

Российские BI-платформы визуализируют свои данные клиента. Не выявлено подтверждённых кейсов полноценного автоматического 360-анализа бизнеса на основе открытых источников среди российских BI-решений. Западные AI-платформы ($4B+ AlphaSense) не работают с российскими реестрами. AI.Инсайт (Контур) ближе всех, но сфокусирован на проверке контрагентов, а не на комплексном стратегическом анализе. BSR — один из немногих продуктов на пересечении: AI + российские реестры + полный 360-анализ.

Автоматизация Due Diligence

В РФ автоматизация DD фрагментарна: Контур.Фокус предлагает автоматические проверки контрагентов через API и ИИ-отчёты, СПАРК — автоматизацию регламентных проверок. Однако комплексный стратегический DD (финансовый + рыночный + операционный + HR) остаётся ручным. В мире: Kira (Litera, заявлена поддержка «any language»), Harvey (десятки языков), Spellbook (140+ языков) — мультиязычные, но не интегрированы с российскими реестрами. По данным McKinsey, AI сокращает трудозатраты в отдельных procurement-активностях на 50-80% ОЦЕНКА. BSR может стать первой платформой комплексного автоматизированного стратегического DD в России.

Каналы продвижения конкурентов

Стратегия лидеров: «бесплатное исследование → цитируемость и экспертность → дорогие заказы». Data Insight публикует бесплатные отчёты по e-commerce → их цитируют все СМИ → бренды заказывают кастомные исследования. Эта модель работает и для BSR: публиковать демо-отчёты → наращивать авторитет.


Каналы: конференции/спикерство (60%), публикация бесплатных отчётов (50%), Telegram-каналы (40%), SEO (30%), referrals (70%).

IX. SWOT-анализ BSR

Позиционирование BSR относительно 62 конкурентов

STRENGTHS

  • Скорость — ~90 секунд vs 2-8 недель у конкурентов
  • Полнота — 25+ секций, 11 этапов, единый отчёт (финансы + HR + digital + конкуренты + стратегия)
  • Типизация — 6 типов бизнеса с адаптивными метриками
  • Calc-trace — прозрачность вычислений (ФАКТ/РАСЧЁТ/ОЦЕНКА)
  • Себестоимость — ~500-2000 руб. за отчёт vs 100K-5M руб.
  • Локализация — ФНС, ЕГРЮЛ, HH.ru, 2ГИС, Rusprofile
  • AI Board — самокритика и доработка отчёта AI-экспертами

WEAKNESSES

  • Качество 5/10 — главная проблема; не конкурирует с ручным консалтингом по глубине
  • Галлюцинации LLM — оценки без обоснования, неверные данные
  • Нет проприетарных данных — все источники публичные
  • Команда 1 человек — нет экспертов, маркетолога, sales
  • Нет платящих клиентов — нет обратной связи от рынка
  • LLM-зависимость — цены OpenAI/Google влияют на unit economics

OPPORTUNITIES

  • Вакуум после Big4/Big3 — рынок консалтинга +15%, спрос > предложение
  • ~6,4 млн МСП (реестр ФНС, 2025) — 95% не могут позволить консалтинг за 500K+
  • AI SaaS растёт двузначными темпами ОЦЕНКА (Allied Market Research: global self-service BI CAGR ~15%; Smart Ranking: рынок AI в РФ +45% в 2024) — готовность рынка к AI-аналитике растёт
  • Банковский скоринг — экспресс-аналитика для кредитования МСП
  • Экосистемы — 1С, Битрикс24, amoCRM как точки входа
  • СНГ — Казахстан, Узбекистан, Беларусь
  • Подписочная модель — мониторинг конкурентов = recurring revenue

THREATS

  • Вход Сбер/Яндекс/Контур — GigaChat + СБИС, AI.Инсайт уже запущен
  • Регуляторные риски — 152-ФЗ, ограничения scraping, закрытие API
  • Репутационный — один скандал с ложными данными = смерть бренда
  • Консалтинг + AI — гибрид «эксперт + AI» может быть сильнее чистого AI
  • ChatGPT бесплатно — «зачем платить, если нейронка и так ответит?»
  • Спад МСБ — выручка +8% при инфляции 9,5% = реальное снижение

XI. Стратегический Roadmap BSR

ТОП-5 приоритетов для выхода на рынок

1

Качество данных: 5/10 → 7/10

Подключить СБИС API, API Rusprofile. Разделить «данные» (из API/scraping) и «анализ» (LLM). Верификация каждого факта по 2+ источникам. Снижение галлюцинаций. Метрика: AI Board score 7.0+ стабильно.

2

Первые 10 платящих клиентов

Ручной outreach к 20-30 предпринимателям. Ценовой тест: 3K / 7K / 15K руб. Сбор обратной связи по 5 вопросам. Метрика: 10 платящих за 2 месяца, NPS > 30.

3

Мониторинг конкурентов (подписка)

Ежемесячный дайджест: изменения финансов, новые вакансии, digital-активность. Telegram-бот + email. Цена: 2-5K руб./мес. Метрика: retention > 60% через 3 месяца.

4

API и B2B-интеграции

API для банковского скоринга. Виджет для 1С/Битрикс24. White-label для консалтинговых фирм. Метрика: 1 B2B-контракт с MRR > 50K руб.

5

Найм ключевого человека

Technical co-founder (AI/ML), или Growth-маркетолог, или отраслевой эксперт-аналитик. Метрика: 1 человек в команде за 3 месяца.
